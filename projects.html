Portfolio Website

February 2021

Using HTML/CSS and JavaScript, I made a portfolio about myself. It includes an “about me” section, a section that highlights my projects, and a resume page that includes a downloadable resume. Also, there is a footer section on each page that includes a link to my LinkedIn and my email address. To create the portfolio, I used various formatting techniques to get my information across. One of those ways included the skeleton formatting for HTML. This formatting helped me put in place the title, contents, and footer of each page. Combined with CSS and JavaScript, it has helped me understand the basics of website development and the frontend side of it.

Biography website of Adele Goldberg

March 2021

I used HTML/CSS to create the biography website of Adele Goldberg. To do this project, however, I had to work with a team. The team had three separate groups. One to gather information on Adele Goldberg, another to design the website, and one person to create the website using HTML/CSS. My role in the process was to create the website using HTML/CSS. This project has taught me how to work in a team setting with website development. The result of my team's information and design has taught me how to communicate with my team, as well as problem-solve to reach our goal.
